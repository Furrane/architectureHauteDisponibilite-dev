Description du projet


Un site web de gestion de contenus hautement disponible à l'adresse www.********. Nous avons choisi de construire ce projet suivant les principes de la methode agile. Découper le projet en étapes consécutives finalisées et facilement réplicables. L'objectif est de fournir au client une solution viable à chaque étape du développement.


Contraintes techniques




Première Etape : Réaliser un site web fonctionnel


Choix du CMS

Le choix du CMS c'est porté sur WORDPRESS. Il s'agit dún CMS gratuit et open-source. Il couvre 70% du marché et est simple d'utilisation. Ce dernier point étant un pré requis du client.

Installation des scripts

Les procédures d'installation ont vocation à être automatisée via le script d'installation
Le pas à pas est conservé ici à des fins de documentation et de maintenance.
